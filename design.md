# Conduit AI - Premium Design System

## Design Philosophy

### Color Palette
- **Primary White**: #FFFFFF - Pure, clean backgrounds
- **Soft Gray**: #F8F9FA - Subtle section backgrounds
- **Medium Gray**: #6C757D - Secondary text and borders
- **Deep Gray**: #343A40 - Primary text
- **Accent Blue**: #0D6EFD - Interactive elements and CTAs
- **Light Blue**: #E7F3FF - Hover states and highlights
- **Success Green**: #198754 - Success states
- **Warning Orange**: #FD7E14 - Warning states
- **Error Red**: #DC3545 - Error states

### Typography
- **Display Font**: Manrope (700, 600, 500 weights)
- **Body Font**: Manrope (400, 300 weights)
- **Accent Font**: Noto Serif Ethiopic (for special headings)
- **Monospace**: JetBrains Mono (for code and data)

### Visual Language
- **Minimalist Apple-inspired**: Clean lines, generous whitespace, surgical precision
- **Depth through shadows**: Subtle elevation with soft shadows
- **Rounded corners**: 12px radius for cards, 8px for buttons
- **Consistent spacing**: 8px base unit system
- **High contrast**: 4.5:1 minimum ratio for accessibility
- **No gradients**: Solid colors only, subtle when needed

## Visual Effects & Styling

### Used Libraries
- **Anime.js**: Smooth micro-interactions and page transitions
- **ECharts.js**: Clean data visualization with custom themes
- **Pixi.js**: Subtle particle effects for hero backgrounds
- **Splide.js**: Premium carousel experiences
- **Matter.js**: Physics-based animations for workflow nodes
- **p5.js**: Creative coding for background effects

### Animation Effects
- **Text Effects**: 
  - Character-by-character reveal for headings
  - Subtle color pulse on interactive elements
  - Smooth fade-ins with stagger delays
- **Background Effects**:
  - Floating geometric particles with physics
  - Subtle parallax scrolling (max 8% translation)
  - Organic noise patterns for texture

### Header Effect
- **Clean Navigation**: Fixed header with backdrop blur
- **Logo Animation**: Subtle scale and opacity transitions
- **Tab Indicators**: Smooth underline with easing
- **Search Expansion**: Animated search bar with focus states

### Interactive Elements
- **Button Hover**: Lift with shadow expansion and scale
- **Card Hover**: Gentle lift with border highlight
- **Toggle Switches**: Smooth slide with spring physics
- **Chat Bubbles**: Typing animation with realistic timing
- **Workflow Nodes**: Physics-based connection animations

### Layout Principles
- **Grid System**: 12-column responsive grid with gutters
- **Vertical Rhythm**: Consistent baseline grid (24px)
- **Content Hierarchy**: Clear visual weight through size and color
- **Breathing Room**: Generous padding and margins throughout
- **Mobile First**: Progressive enhancement for larger screens

### Component Styling
- **Cards**: Clean white backgrounds with subtle shadows
- **Forms**: Minimal styling with clear focus states
- **Modals**: Centered overlay with backdrop blur
- **Navigation**: Clean tabs with active state indicators
- **Data Tables**: Alternating row colors with hover states

### Dashboard Specific
- **Status Indicators**: Color-coded with iconography
- **Metrics Cards**: Large numbers with descriptive labels
- **Workflow Nodes**: Connected flow with physics
- **Data Visualization**: Custom ECharts theme matching design
- **Activity Feeds**: Timeline with clear visual hierarchy

### Mobile Considerations
- **Touch Targets**: Minimum 44px for all interactive elements
- **Collapsible Sections**: Smooth accordion animations
- **Swipe Gestures**: Horizontal navigation for workflows
- **Bottom Navigation**: Fixed mobile navigation bar
- **Optimized Typography**: Larger text sizes for readability

### Premium Features
- **Dark Mode Support**: Automatic system preference detection
- **Reduced Motion**: Respects user accessibility preferences
- **High Contrast Mode**: Enhanced visibility option
- **Keyboard Navigation**: Full keyboard accessibility
- **Screen Reader Support**: Proper ARIA labels and semantics

This design system creates a cohesive, premium interface that rivals the best in class applications while maintaining exceptional usability and accessibility.
